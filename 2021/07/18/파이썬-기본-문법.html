  1. 식별자
    • 변수, 함수, 클래스, 모듈 등의 이름을 식별자라고 합니다.
    • 식별자 규칙
    • 소문자, 대문자, 숫자, _(언더스코어)를 사용
    • 가장 앞에 숫자 사용 불가 ex) 10a = 5 << 안됨
    • 예약어의 사용 불가 : def, class, try, except ...

      • snake case : fast_campus : 변수선언, 함수 선언할 때 사용
